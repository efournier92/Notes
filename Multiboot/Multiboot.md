#### Expand

```bash

sudo mount /dev/$part_id /mnt/slot
iso_file=''
sudo mkdir -p /mnt/tmp
sudo mount -o loop $iso_file /mnt/tmp
sudo cp -rv /mnt/tmp/* /mnt/slot
```
